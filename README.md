# Website Portofolio

Ini adalah situs portofolio statis yang dibuat dengan HTML, CSS, dan JavaScript.

## Fitur

- Navigasi responsif
- Tampilan beranda, tentang, layanan, proyek, testimoni, dan kontak
- Formulir kontak dengan validasi sederhana
- Animasi reveal saat scroll
- Desain modern dan mobile-friendly

## Cara Menjalankan

1. Buka `index.html` langsung di browser.
2. Atau gunakan server lokal sederhana, misalnya dengan Python:

```bash
cd /workspaces/sodikin
python3 -m http.server 8000
```

Lalu buka http://localhost:8000 di browser.

## Kustomisasi

- Ganti nama, email, dan konten pada `index.html`.
- Sesuaikan warna dan tipografi di `styles.css`.
- Tambahkan proyek atau testimoni sesuai kebutuhan.
